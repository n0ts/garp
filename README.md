Usage:
  garp -i eth0 -a 192.168.1.1 -n 5
  garp -h
  garp -v

Commands:
Either long or short options are allowed.
  garp --interface       -i       Send gratuitous arp using the interface.
  garp --use-file        -f       Use the specified file containing set of IP.
  garp --ipaddress       -a       Use the specified IP address.
  garp --arp-num         -n       Send specified number of gratuitous ARP.
  garp --verbose         -V       Use verbose output infos.
  garp --help            -h       Display this short inlined help screen.
  garp --version         -v       Display the version number
